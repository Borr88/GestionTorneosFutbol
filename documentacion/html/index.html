🗄️ Modelos (Models)

models.db_manager
Gestor de base de datos MySQL. Maneja todas las operaciones CRUD para equipos, participantes, partidos y estadísticas. Implementa la lógica de negocio del torneo.

🎮 Controladores (Controllers)

controllers.main_controller
Controlador principal que gestiona la ventana y navegación entre vistas. Coordina el tablero del torneo y todos los sub-controladores.
controllers.equipo_controller
Gestiona el registro y edición de equipos. Permite crear, modificar y eliminar equipos del torneo.
controllers.participante_controller
Controla el registro de participantes/jugadores. Gestiona datos personales, estadísticas y asignación a equipos.
controllers.listado_equipos_controller
Muestra el listado completo de equipos registrados con opciones de búsqueda y filtrado.
controllers.listado_participantes_controller
Lista todos los participantes con funcionalidad de edición, eliminación y visualización de estadísticas.
controllers.enfrentamiento_controller
Gestiona los enfrentamientos entre equipos. Permite crear partidos y registrar resultados.
controllers.eventos_partido_controller
Registra eventos individuales durante los partidos: goles, asistencias, tarjetas, etc.
controllers.calendario_controller
Muestra el calendario de partidos organizados por fechas y fases del torneo.
controllers.creditos_controller
Vista "Acerca de" con información del desarrollador, versión y tecnologías utilizadas.
controllers.guia_controller
Manual de usuario en HTML con botón para abrir la versión PDF completa.
